# Chrome Web Store Privacy Policy

## 用户诉求

推进 Open Browser Use Chrome Web Store 正式发布，并补齐商店后台所需的发布资料。

## 改动

- 新增 `docs/PRIVACY_POLICY.md`，作为 Chrome Web Store 隐私政策 URL。
- 隐私政策说明扩展可能处理的数据类别、本地 native host 数据流、用途限制、存储方式和支持入口。

## 设计动机

Chrome Web Store 在扩展声明会访问用户数据时要求提供公开隐私政策。Open Browser Use 具备 tabs、debugger、history、downloads、scripting 和 `<all_urls>` 等能力，隐私政策必须如实覆盖网页内容、历史记录、认证信息和其他页面中可能出现的敏感数据，而不能缩小披露范围。
